# Data

Large raw and processed datasets are not stored directly in this GitHub repository.

The dataset and reproducible notebook will be uploaded to Kaggle and linked here after publication.

The study uses:

- stock-related tweets
- historical stock-price records
- Financial PhraseBank records
- processed ticker-date sentiment features
- technical indicators
- final modeling table
- experimental output files
